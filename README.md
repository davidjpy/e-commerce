# :shopping_cart: WildBear
Wildbear is e-commerce website designed for selling camping gear.

## Stacks
* ReactJS
* Redux Toolkit & Redux Toolkit Query
* Django
* Netlify
* Postgresql
* Amazon Web Services - Elastic Beanstalk, EC2, RDS, Load Balancers, ACM & Route 53

## Demonstration
